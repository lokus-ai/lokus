Hey r/selfhosted,

I've been working on Lokus, an open-source note-taking app, and wanted to share it with the community that would probably appreciate it most.

**What is it?**

A local-first note-taking app with wiki links, graph view, and a powerful template system. Think Obsidian but fully open source.

**Why I built it:**

I was frustrated that:
- Obsidian's sync costs $10/month
- Notion stores everything on their servers
- Most alternatives are either Electron-bloated or lack features

**Tech stack:**
- Tauri + Rust backend (not Electron)
- React + TipTap editor
- All notes are plain markdown files â€” no database, no SQLite

**Features:**
- Wiki links with fuzzy autocomplete
- Graph view for knowledge visualization
- 18 task states (todo, in-progress, urgent, delegated, etc.)
- Infinite canvas for visual thinking
- 40+ slash commands
- 90+ template features (variables, conditionals, date math)
- Full theming support

**What's coming:**
- P2P sync using Iroh (in development)
- Plugin system with MCP integration

**Self-hosting:**

Right now it's a desktop app (Mac/Windows/Linux) with local files. The P2P sync will let you sync between your own devices without any central server.
