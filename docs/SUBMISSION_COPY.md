## üìÑ Long Description (500 words)

```
# What is Lokus?

Lokus is a next-generation note-taking app that combines the best features of Notion and Obsidian while addressing their key limitations. It's built for developers, writers, researchers, and knowledge workers who want powerful database features without sacrificing speed, privacy, or control over their data.

## Key Features

**Database Views (Like Notion, Built-In)**
Transform your markdown notes into Notion-style databases with 8 property types (text, number, date, select, multi-select, checkbox, URL, email). No plugins required. Use YAML frontmatter as database properties, with inline editing, advanced filtering (AND/OR logic, 15+ operators), and multi-column sorting.

**Interactive Knowledge Graphs**
Visualize your notes as interactive 2D and 3D graphs with community detection, tag filtering, and real-time updates. Navigate by clicking nodes, export as PNG/SVG, and handle 10,000+ notes with sub-100ms rendering.

**AI Integration (MCP Server)**
Built-in Model Context Protocol server with 68+ tools for note management, workspace operations, advanced search, content analysis, file operations, and editor enhancements. Connects to any AI assistant via MCP protocol. Auto-starts with Lokus - zero configuration needed.

**Quantum Search Architecture**
100x faster than traditional search engines. Quantum Superposition Index provides O(1) lookups, Neural Semantic Cache enables predictive search. Benchmarks: 10,000 files searched in 22ms vs 2,400ms in alternatives, with 90% less memory usage.

**Rich Markdown Editor**
Industry-leading TipTap editor with 100+ languages syntax highlighting, LaTeX math support (inline and block), wiki links with autocomplete, resizable tables, task lists with multiple statuses, lazy-loading images, smart paste (HTML ‚Üí Markdown), split pane editing, and optional Vim mode.

**Canvas & Kanban**
Infinite canvas for freeform spatial thinking powered by TLDraw, plus visual Kanban boards for task management.

## Why Choose Lokus?

**Performance:**
- 10MB app size (vs 100MB in Electron-based alternatives)
- <1 second startup time
- ~30MB memory usage (vs ~300MB)
- 100x faster search

**Privacy & Control:**
- 100% local-first - all data stays on your device
- No cloud required (use any sync service you want)
- No telemetry, tracking, or analytics
- No vendor lock-in - just markdown files

**Compatibility:**
- Obsidian compatible - point it at your existing vault
- Standard markdown format
- Works with any cloud sync (Dropbox, Google Drive, iCloud, Syncthing)

**Open Source:**
- MIT License - completely free
- Active development and community
- Transparent roadmap
- Contribution-friendly codebase

## Who Is It For?

- **Developers** who want fast, local-first knowledge management
- **Writers** who need rich formatting and organization
- **Researchers** managing large collections of notes
- **Students** organizing class notes and projects
- **Anyone** who values privacy and owns their data

## Tech Stack

Built with modern, performant technologies:
- Frontend: React 19, TipTap 3.4, Tailwind CSS, Vite
- Backend: Rust, Tauri 2.0, Tokio (async runtime)
- Graph: Three.js, Sigma.js, react-force-graph
- Canvas: TLDraw 2.0
- Math: KaTeX 0.16+
- Testing: Vitest + Playwright (500+ tests)

## Cross-Platform

Available for Windows, macOS, and Linux with native performance on each platform.

## Getting Started

1. Download from GitHub releases
2. Install and open Lokus
3. Point it at your existing notes or create a new workspace
4. Start writing!

No account needed. No cloud setup. No subscriptions. Just download and use.

## Roadmap

- Mobile apps (iOS & Android) - Q1 2026
- Calendar view for databases
- Formula support (spreadsheet-like calculations)
- PDF annotations
- Web clipper extension
- E2E encryption (optional)

100% free. Forever. Open source.
```

**Q: How is this different from Notion?**
A: Lokus is local-first (all data on your device), open source, and free. No cloud required. You own your data. It's also 100x faster and works offline.

**Q: Can I import my Obsidian vault?**
A: Yes! Just point Lokus at your existing Obsidian folder. Everything works - notes, links, attachments. No migration needed.

**Q: Is sync really free?**
A: Yes! Use any cloud provider (Dropbox, Google Drive, iCloud, Syncthing). No vendor lock-in.

**Q: What about mobile apps?**

**Q: Can I contribute?**
